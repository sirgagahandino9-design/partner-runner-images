# Changelog

All notable changes to this project will be documented in this file.

## [Archived] - 2026-06-12

### ⚠️ IMPORTANT: Repository Archived

This repository has been archived as GitHub has assumed ownership of Arm64 runner images.

#### What's Happening

- **Archive Date:** June 12, 2026
- **New Location:** [actions/runner-images](https://github.com/actions/runner-images)
- **Previous Partner:** Arm Limited, Inc.

#### User Impact

✅ **No Impact on Existing Workflows**
- Your existing workflows using `ubuntu-24.04-arm`, `ubuntu-22.04-arm`, and `windows-11-arm` will continue to work

📋 **What's Moving**
- All future development will occur in [actions/runner-images](https://github.com/actions/runner-images)
- Issues from this repository are being migrated
- Bug reports should be filed in the new repository

#### Migration Guide

**If you have open issues:**
1. They will be automatically migrated to [actions/runner-images](https://github.com/actions/runner-images)
2. No action needed from you

**If you need to file a new issue:**
1. Go to [actions/runner-images](https://github.com/actions/runner-images)
2. Create an issue there
3. Reference this repository if needed for context

**For feature requests:**
- Submit to [actions/runner-images](https://github.com/actions/runner-images)
- Include details about your use case
- Specify which image(s) you're using
