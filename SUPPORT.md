# Support

## ⚠️ Repository Archival Notice

**This repository (`actions/partner-runner-images`) has been archived as of June 12, 2026.**

GitHub has assumed ownership of Arm64 runner images. All future development, bug fixes, and feature requests will be handled in the main GitHub Actions repository.

---

## Where to Get Help

### 🔴 For Issues with Arm-based Images

**Please report all issues and requests here:**
- 👉 **[actions/runner-images](https://github.com/actions/runner-images)**

This is where all GitHub Actions runner images are now maintained, including Arm-based images.

### ✅ Current Images Still Available

Your existing workflows using the following images will continue to work:
- `ubuntu-24.04-arm` (Ubuntu 24.04 Arm64)
- `ubuntu-22.04-arm` (Ubuntu 22.04 Arm64)
- `windows-11-arm` (Windows 11 Arm64)

### 📚 Documentation

For comprehensive information about:
- Available tools and software on each image
- Image specifications and versions
- Supported use cases

Visit: https://github.com/actions/runner-images/tree/main/images

### 🐛 Reporting Bugs

When reporting bugs related to Arm-based runner images:

1. Go to https://github.com/actions/runner-images/issues
2. Click "New Issue"
3. Include details about:
   - Which image you're using (`ubuntu-24.04-arm`, `ubuntu-22.04-arm`, or `windows-11-arm`)
   - Your GitHub Actions workflow YAML
   - Error messages or unexpected behavior
   - Steps to reproduce the issue

### 💡 Requesting Features

To request new tools or software on Arm-based images:

1. Visit https://github.com/actions/runner-images/issues
2. Search existing issues first to avoid duplicates
3. Create a new issue describing:
   - What tool or software you need
   - Why it's important for your workflows
   - The specific image(s) where you need it

### 📞 GitHub Support

For account or billing questions, or if you need additional assistance:
- **GitHub Support Portal:** https://support.github.com
- **GitHub Community:** https://github.com/orgs/community/discussions

### ℹ️ About This Archive

This repository served as a temporary home for Arm-based runner images while they were in partnership. Now that GitHub has full ownership and integration with the main Actions infrastructure, all related development has consolidated into the primary `actions/runner-images` repository.

## FAQ

**Q: Will my existing workflows stop working?**
A: No, your workflows will continue to work with the existing Arm-based images.

**Q: Where should I report bugs with Arm images?**
A: Please report them in the [actions/runner-images](https://github.com/actions/runner-images) repository.

**Q: Can I still request new tools on Arm images?**
A: Yes, please create an issue in [actions/runner-images](https://github.com/actions/runner-images) with your request.

**Q: What happens to issues reported here?**
A: Issues from this repository are being migrated to the new repository.

**Q: When was the archival date?**
A: This repository was archived on June 12, 2026.
